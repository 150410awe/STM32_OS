### 代码风格

- **头文件保护**：使用 `#pragma once`
- **注释风格**：采用Doxygen风格，提供详细的中文注释
- **命名约定**：
  - 全写：使用下划线分隔 + 小写字母（如：`memory_distribution`）
  - 简写：使用下划线分隔 + 大写字母（如：`FSMC_address`）

### 项目(我)设计原则

1.**类型安全**
2.**编译期优化**
3.**模块化设计**
4.**分层设计**
5.**可扩展**
6.**内部复杂外部简单**
7.**代码即文档**
8.**有安全性和可靠性,但同时不应有太多复杂性,还有冗余**
